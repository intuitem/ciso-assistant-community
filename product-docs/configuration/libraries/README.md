# Libraries

Libraries are how content gets into CISO Assistant — frameworks, threats, risk matrices, reference controls, mappings. They ship as YAML files, are versioned, and can be upgraded or removed cleanly.

This section covers everything you need to manage them on your instance:

- [Designing your own libraries](custom-libraries.md) — the Excel-to-YAML authoring workflow.
- [Getting your custom framework](custom-frameworks.md) — quick-start for adding a single framework.
- [CIS Controls / Cloud Controls Matrix (CCM)](cis-controls.md) — direct import of CIS/CSA spreadsheets.
- [Library upgrade](library-upgrade.md) — pull a newer version of an already-loaded library.
- [Upgrading a library](upgrading-a-library.md) — refresh built-in libraries after updating your instance.
- [Library clean-up](library-cleanup.md) — remove a loaded library when it's no longer in use.

See also: [Libraries concept](../../concepts/libraries.md), [Frameworks concept](../../concepts/frameworks.md).
